## Installation

### *Step 1: run the commands below in your terminal*

```
  git clone https://github.com/awmirs/django-utubedl
  cd django-utubedl
  python3 manage.py runserver
```
### *Step 2: open the url in your browser*
```
http://127.0.0.1:8000/
```
